# SaúdeGlobal 🌍💉

Portal informativo sobre doenças, pandemias, desastres naturais e análises estatísticas com gráficos interativos.

## Tecnologias Utilizadas

- Frontend: React.js
- Backend: Node.js + Express (em breve)
- Banco: Firebase
- Mapas: Leaflet.js
- Gráficos: Chart.js
- Exportação: Excel
